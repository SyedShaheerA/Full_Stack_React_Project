# HomeAroma - Food Discovery Platform

HomeAroma is a modern, responsive web application built with React and Bootstrap. It provides a seamless user experience for discovering food options, featuring a dynamic landing page and a dedicated user authentication system.

## 🚀 Features

* **User Authentication:** Fully integrated Sign Up and Login screens.
* **Responsive Design:** Built using Bootstrap 5 for a mobile-first approach.
* **Dynamic UI:** Includes a high-impact search carousel on the landing page for quick navigation.
* **Client-Side Routing:** Managed via `react-router-dom` for fast, no-refresh page transitions.
* **Dark Mode Support:** Integrated with `bootstrap-dark-5` for a modern aesthetic.

## 🛠️ Tech Stack

* **Frontend:** React.js (Hooks, Functional Components)
* **Styling:** CSS3, Bootstrap 5, Bootstrap-Dark-5
* **Routing:** React Router v6
* **Testing:** Jest, React Testing Library
